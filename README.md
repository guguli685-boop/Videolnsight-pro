### VideoInfoViewer 



### 介绍

VideoInfoViewer 它能够快速读取视频文件的各种属性，包括格式、编码、分辨率、时长、音频参数等，同时提供缩略图生成、插件扩展、主题切换等多项实用功能，适用于视频创作者、媒体工作者、内容审核人员以及任何需要查看视频详细信息的用户。

### 核心功能特点

🎬 视频信息分析
自动解析视频格式、分辨率、帧率、时长、编码方式等技术参数

同时支持音频信息读取（声道、采样率、比特率等）

🖼️ 视频缩略图生成
支持从任意视频中自动截取缩略图

可自定义缩略图大小与圆角效果，使预览更美观

🎨 主题与界面模式
自适应系统浅色/深色主题

支持手动切换主题

内置“简洁模式”可隐藏工具栏、状态栏与文件列表，实现全屏沉浸式体验

🔌 插件扩展能力
支持基于 Python 的插件系统
用户可自由扩展功能，也可使用社区插件
